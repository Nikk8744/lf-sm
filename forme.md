# Local Farmers' Marketplace

### Tech Stack
- Frontend: Next.js (App Router), Tailwind CSS, React Query.
- Backend: Next.js API routes, Drizzle ORM, PostgreSQL (Neon).
- Auth: NextAuth.js (Google OAuth), JWT.
- Validation: Zod.
- Payments: Stripe.
- Realtime: WebSocket or SSE.
- Deployment: Vercel.
